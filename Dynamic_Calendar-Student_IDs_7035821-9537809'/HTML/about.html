Description

Thy Dynamic Calendar is a web application is a project which was part of Web-Engineering 1 course of our university. It has all the basic and enhanced features listed below.
DHBW Stuttgart

Basic Fearures

  • Show calendar entries
  • Create calendar entry
  • Edit calendar entries
  • Delete calendar entries
  • Image for calendar entry

Enhanced Features

  • Alerts with sound (sound needs to be enabled by the user)
  • Back button handling on all pages
  • Show categories from server
  • Extra pages for editing and adding entries and categories
  • Image preview on the edit/add entry page (implemented with canvas)
  • Refresh button handling on the edit/add entry page
  • About page
  • Contact page with contact editEntryForm and php script

Further Notes

We created a php script for sending the message in the contact editEntryForm directly to us. However, since we don't have a server, we were not able to test it. As a compromise, your local e-mail editor is opened when a contact editEntryForm is submitted. All fields are pre-filled with the message from the contact editEntryForm.

Furthermore, we tested our web application on the following browsers which seemed to have no major issues: Firefox, Microsoft Edge and Safari on an iOS device.
